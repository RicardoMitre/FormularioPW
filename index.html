Perguntas

  1. O atributo placeholder especifica uma pequena dica que descreve o valor esperado de um campo de entrada.A dica curta é exibida no campo de entrada antes que o usuário insira um valor.

  2. Um atributo booleano que, se presente, indica que este botão de opção é o padrão selecionado no grupo.

  3. O atributo value é compartilhado por todos os tipos da tag input; no entanto, serve a um propósito especial para entradas do tipo radio: quando um formulário é enviado, apenas os botões de rádio que estão atualmente marcados são enviados ao servidor e o valor relatado é o valor do atributo value. Se value não for especificado de outra forma, será a string onpor padrão.

  4. Um grupo de rádio é definido dando a cada um dos botões de rádio no grupo o mesmo name.

  5. A tag label representa uma legenda para um item em uma interface de usuário. O elemento for relaciona a tag label com o ID da teg input.
